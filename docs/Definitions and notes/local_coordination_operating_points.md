# Parallel Project / Silo Map

The point system describes the **communication layer** of the LCP. It names things, routes people to things, lets things receive logs, and gives each thing a page or interface.

Separate but connected systems should be described as parallel silos:

1. Communication / Point System
2. House Operations System
3. Financial System
4. Labor Pool System
5. EEU / Internal Unit System
6. Market / Exchange System
7. Intelligence / History System
8. Game Layer

## Current Working Interpretation

```text
Point system = communication and addressing layer
House operations = real-world management layer
Financial system = money and capital layer
Labor pool = work execution layer
EEU = internal value/accounting layer
Market = offer/request/exchange layer
Intelligence/history = memory and understanding layer
Game = optional participation/play layer on top
```

## Next Recommended Silo to Define

Start with **House Operations System** because it is the most concrete and connects houses, rooms, people, tasks, calendar, documents, labor, and money.
